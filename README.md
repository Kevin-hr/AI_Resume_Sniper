# 🎯 AI Resume Sniper (AI 简历狙击手)

> **版本**: v1.1 | **状态**: MVP (Ready for Demo) | **作者**: Digital Life Kha'Zix

## 1. 项目概述 (Overview)

**AI Resume Sniper** 是一个基于大语言模型（LLM）的自动化简历优化系统。它模拟一位拥有 10 年经验的资深 HRBP（人力资源业务合作伙伴）的视角，在 60 秒内对候选人的简历进行深度审计。

### 核心价值
*   **模拟 6 秒筛选**：模拟真实 HR 的快速筛选过程，预测简历是否会被秒拒。
*   **ATS 通过率优化**：识别并填充职位描述（JD）中的关键关键词，确保通过 ATS 系统。
*   **STAR 法则重写**：自动将平淡的工作描述重写为高价值的 STAR（情境-任务-行动-结果）格式。
*   **商业化资产**：设计为“零边际成本”的 SaaS 资产，支持自动化运行和分层付费模式。

### 技术栈 (Tech Stack)
*   **核心逻辑**: Python 3.9+
*   **Web 界面**: Streamlit (快速 MVP 构建)
*   **AI 引擎**: DeepSeek-V3 (通过 OpenAI 兼容接口调用)
*   **项目架构**: 模块化设计 (Src/Docs/Data/Tests)

## 2. 安装指南 (Installation)

### 系统要求
*   Windows / macOS / Linux
*   Python 3.9 或更高版本
*   DeepSeek API Key (或其他兼容 OpenAI SDK 的 LLM Key)

### 依赖安装
建议使用虚拟环境进行安装：

```bash
# 1. 克隆项目 (假设已下载)
cd AI_Resume_Sniper

# 2. 创建虚拟环境 (可选)
python -m venv venv
# Windows 激活:
.\venv\Scripts\activate
# Mac/Linux 激活:
source venv/bin/activate

# 3. 安装依赖
pip install openai streamlit python-dotenv
```

### 环境配置
1.  在项目根目录 (`AI_Resume_Sniper`) 创建 `.env` 文件。
2.  添加 API Key：
    ```text
    DEEPSEEK_API_KEY="sk-your-key-here"
    ```

## 3. 使用说明 (Usage)

### 快速启动 (Web UI)
启动可视化界面，进行交互式简历诊断：

```bash
streamlit run src/web_ui.py
```
*   浏览器自动打开 `http://localhost:8501`
*   左侧输入简历内容，右侧输入 JD 内容，点击 **🚀 Analyze Resume**。

### 命令行测试 (CLI)
运行集成测试脚本，生成演示报告：

```bash
python tests/integration_test.py
```
*   输入文件：`data/sample_resume_A.md`, `data/sample_jd_A.md`
*   输出报告：`data/sample_report_A.md`

### 功能模块详解
1.  **Match Score (匹配打分)**: 0-100 分，<60 分建议直接放弃或重写。
2.  **Fatal Red Flags (致命红旗)**: 找出 HR 拒信的 3 个核心原因。
3.  **STAR Rewrite (高光重写)**: **[付费核心]** 将一段经历重写为数据驱动的成就。
4.  **Quick Fixes (速效救心丸)**: 3 个立竿见影的修改建议。
