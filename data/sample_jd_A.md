# 🎯 Target Job Description (JD) - AI Product Manager

**职位名称**: AI产品经理 (B端 SaaS方向)
**薪资范围**: 30k-50k · 15薪

## 岗位职责 (Responsibilities)
1.  **产品规划**: 负责公司核心AI产品的规划与设计，能够独立完成高质量的 PRD 文档，推动产品从 0 到 1 落地。
2.  **场景落地**: 深入理解大模型 (LLM) 技术边界，探索 RAG、Agent 等技术在 B 端业务流中的应用场景。
3.  **数据驱动**: 建立产品数据指标体系，通过数据分析驱动产品迭代，提升用户留存率和商业转化率。
4.  **跨部门协作**: 协调算法、研发、设计及运营团队，确保项目按时高质量交付。

## 任职要求 (Requirements)
1.  **经验要求**: 3年以上互联网产品经理经验，至少 1 年 AI/LLM 相关产品经验，有 B 端 SaaS 成功案例者优先。
2.  **硬性技能**: 熟练掌握 Axure/Figma 等原型工具，具备极强的逻辑思维能力和文档撰写能力。
3.  **技术理解**: 熟悉 RAG、Fine-tuning、Prompt Engineering 等技术原理，能与算法工程师无障碍沟通。
4.  **加分项**: 有技术背景 (开发转产品) 或 创业经历者优先。
